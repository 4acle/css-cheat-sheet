CSS Selectors & Specificity Cheat Sheet

Basic Selectors

Selector Name Formatted Selector Usage Description
Universal * Selects all elements
Element p Selects elements by tag name
Class .class Selects elements by class
ID #id Selects element by ID

Advanced Selectors

Selector Name Formatted Selector Usage Description
Attribute [type="text"] Selects input elements with type attribute set to text
Pseudo-class :hover Selects elements when hovered over
Pseudo-class :nth-child(2) Selects the second child of its parent
Negation :not(.active) Selects all elements that do not have the class \"active\"
Universal with attribute *[lang] Selects all elements with a lang attribute
Child combinator ul > li Selects direct li children of a ul
Adjacent sibling h1 + p Selects the first p immediately after an h1

Specificity Calculation

Specificity order: Inline Style > ID > Class/Pseudo-class > Element

  • Inline styles: 1000
  • ID selectors (#id): 0100
  • Class selectors (.class): 0010
  • Element selectors (div): 0001
