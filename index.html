CSS Selectors & Specificity Cheat Sheet

CSS Selectors

Selector Description Example
* Selects all elements * { margin: 0; }
element Selects elements by tag name> p { color: blue; }
.class Selects elements by class name .highlight { background-color: yellow; }
#id Selects elements by ID #main { padding: 20px; }

Combinators

Combinator Description Example
element descendant Selects descendant elements div p { color: green; }
element > child Selects direct child elements ul > li { margin-left: 20px; }
element + sibling Selects adjacent sibling elements h2 + p { margin-top: 0; }
element ~ sibling Selects sibling elements h2 ~ p { margin-top: 0; }

Pseudo-classes

Pseudo-class Description Example
:hover Selects elements when hovered over a:hover { color: red; }
:nth-child(n) Selects elements based on their position in a parent element li:nth-child(2) { font-weight: bold; }
:first-child Selects the first child element p:first-child { font-weight: bold; }
:last-child Selects the last child element p:last-child { font-weight: bold; }

Specificity Calculation

Specificity order: Inline styles > ID > Class > Element

  • Inline styles: 1000
  • ID selectors (#id): 0100
  • Class selectors (.class), attributes ([attr]), pseudo-classes (:hover): 0010
  • Element selectors (div, p, h1), pseudo-elements (::before): 0001

Example: #nav .menu li.active has specificity 0-1-2-1
